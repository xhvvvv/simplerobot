## 简单聊天机器人

### v0.1 实现简单命令行交互  
用Java语言实现简单的命令行交互聊天机器人
- 结构化程序
将用户输入语句中的"你"改成"我", "?"改成"!"， “吗”移除 返回

示例
~~~text
在吗？
在！
你好
你好
能听懂汉语吗？
能听懂汉语！
真的？
真的！
English too？
English too！
bye
bye
~~~

好啊，请告诉我你的年龄分别被3，5，7整除的余数

0
